Lab 1.2: Pseudocode

1. คำนวณพื้นที่วงกลม

BEGIN
    INPUT radius
    SET PI = 3.14159
    SET area = PI * radius * radius
    OUTPUT "Area of the circle is: " + area
END

2. คำนวณเกรด

BEGIN
    INPUT score
    IF score >= 80 THEN
        OUTPUT "A"
    ELSE IF score >= 70 THEN
        OUTPUT "B"
    ELSE IF score >= 60 THEN
        OUTPUT "C"
    ELSE IF score >= 50 THEN
        OUTPUT "D"
    ELSE
        OUTPUT "F"
END

3. หาค่าสูงสุดใน 3 จำนวน

BEGIN
    INPUT a, b, c
    SET max = a
    IF b > max THEN
        SET max = b
    IF c > max THEN
        SET max = c
    OUTPUT "Max value is: " + max
END
