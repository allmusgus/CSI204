Lab 1.2: Pseudocode การทำงาน

1. Pseudocode คำนวณพื้นที่วงกลม

BEGIN
    INPUT radius
    SET PI = 3.14159
    SET area = PI * radius * radius
    OUTPUT "Area of the circle is: " + area
END

2. Pseudocode คำนวณเกรด (0-100)

BEGIN
    INPUT score
    IF score >= 80 THEN OUTPUT "A"
    ELSE IF score >= 70 THEN OUTPUT "B"
    ELSE IF score >= 60 THEN OUTPUT "C"
    ELSE IF score >= 50 THEN OUTPUT "D"
    ELSE OUTPUT "F"
END

3. Pseudocode หาค่าสูงสุดใน 3 จำนวน

BEGIN
    INPUT num1, num2, num3
    SET max = num1
    IF num2 > max THEN max = num2
    IF num3 > max THEN max = num3
    OUTPUT "Maximum value is: " + max
END
